# Contributing
You can contribute to this project as your heart desires but please keep note of the following.
- We do not follow any style guide but kindly follow the format found adjacent to the code your are writing.
- When submitting an AI-assisted/generated work, disclose it and make sure you what it is doing line-by-line.
- Please write a good description about your PR, even if it is short as long as it describes the patch.
